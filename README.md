# Math Game

![Example](https://github.com/luique16/math_game/blob/main/img/example.png)

## Description

Each column and row has a specific sum, find and select the correct numbers that make up the sum and delete the incorrect ones.

## How to run

After cloning the repository, run the following command:

```bash
make build && ./math_game

# or 

make run
```

## Dependencies

- [Bubble Tea](https://github.com/charmbracelet/bubbletea)
- [Lip Gloss](https://github.com/charmbracelet/lipgloss)
